安装适用于 Linux 的 Windows 子系统（WSL）

发表于 2021-09-18 更新于 2021-12-17

要安装 WSL，要求 Windows 版本必须符合下面列出的任意一项：

  • Windows 10 或 11
  • Windows Server 2019（版本 1709）或更高版本

Windows 10 或 11

有两种方式安装 WSL：

  • 简化安装：执行命令 wsl --install 即可。但是要求系统版本必须为 Windows 10 2004（内部版本 19041）或更高，或 Windows 11。
  • 手动安装：可以在任意版本的 Windows 10 上安装，但是需要执行多个步骤。

更新到 WSL 2（可选）

检查系统版本

WSL 2 要求操作系统的最低版本为：

  • 对于 x64 系统，需要 1903（内部版本 18362）或更高版本。
  • 对于 ARM64 系统，需要 2004（内部版本 19041）或更高版本。

若要检查系统版本及内部版本号，按下 windows + R 键，然后输入 winver，点击“确定”即可。

启用 虚拟机平台

在 Windows 功能的窗口中，启用 虚拟机平台 功能，然后重启计算机。

或者，使用管理员权限执行命令：

下载 Linux 内核更新包
  • 适用于 x64 系统的 WSL2 Linux 内核更新包
  • 适用于 ARM64 系统的 WSL2 Linux 内核更新包

双击以运行下载的更新包。（系统将提示你提供提升的权限，选择“是”以批准此安装）
